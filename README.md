dotfiles
========

Simple dotfiles repo. Contains a bash script to quickly create symlinks

`antigen` used for zsh
`plugged` used for vim

- .gitconfig
- .slate
- .tmux.conf
- .vimrc
- .zshrc
- antigen.zsh
- plug.vim

```bash
# create symlinks
$ ./bin/dotfiles symlinks

# install tools I like
$ ./bin/dotfiles tools
```
